Coffee Shop Management Application

The system utilises PHP, Bootstrap, and PhpMyAdmin.

Simplifies inventory management by automatically calculating stock levels, costs, and revenue. Integrated secure database handling, enhancing real-time order processing. A streamlined rota system that makes it easy to track employees' hours and automatically calculates wages for time worked.

Admin-Only Features

Enables the admin to view sales data immediately after updating the inventory. Allows the admin to add and remove shifts from the rota for easy updates. Provides the ability to add and remove stock directly from the inventory, automatically calculating costs, revenue, and profit. Uses regular expressions to enable admin accounts based on the sign-up email, offering a secure method for administration management.
