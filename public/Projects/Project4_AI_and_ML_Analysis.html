Data modelling analysis, Ukraine attacks a University AI project

This project explores how machine learning models can be applied to Ukraine's missile attack dataset, with the aim of solving two main challenges: forecasting the number of attacks in the coming month, and identifying the most frequently targeted regions. By leveraging data from a dataset from Kaggle, this project seeks to be used as research for humanitarian, strategic, and research-based decisions.

  • Forecasting Attacks: A time series regression model is used to predict missile and UAV attacks for the next month.
  • Identifying High-Risk Areas: A Random Forest classifier determines which regions have been most consistently targeted since the start of the conflict.
  • Real-World Impact: These insights can assist with evacuation strategies, defence reinforcement, and humanitarian aid allocation.

Skills used in this project include:

  • Data Cleaning: Manipulating data types, removing incorrect or null values, and making adjustments to the dataset to ensure accurate predictions can be made.
  • Data Visualisation: Using libraries such as Matplotlib and Seaborn to visualise the data, making predictions and decision-making easier to understand.
  • Hyperparameter Tuning: Learnt how to use hyperparameter tuning to improve a model's accuracy by optimising hardware usage and maximising training performance.
  • Presentation Skills: Presented findings to my class and accepted constructive criticism to improve the project.
