Java Server chat room Application

This is a university project that is currently in development. Once completed, the application will include the following features:

  • Username creation, allowing others to see their messages associated with that name.
  • Be able to message other active clients on the server.
  • View messages sent by other clients on the server.

The application will also include an admin interface that allows administrators to:

  • Start and stop the server application.
  • Specify the port on which the server runs.
  • Access a log file of server activity.
  • View a web page displaying a list of currently connected users.
